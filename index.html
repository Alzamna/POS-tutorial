Pengenalan Sistem

👨‍💼 Admin Restoran

  • Mengelola kategori menu
  • Menambah & mengedit menu makanan/minuman
  • Memonitor pesanan masuk secara real-time
  • Mengelola meja & kode QR
  • Melihat dashboard & analitik penjualan
  • Perencanaan stok dengan regresi linear

  • Memonitor seluruh restoran terdaftar

  • Melihat menu tiap restoran
  • Restore akun admin yang dinonaktifkan
  • Dashboard ringkasan seluruh sistem
Langkah Awal ADMIN SUPERADMIN
🔐

Login ke Sistem

Cara masuk ke halaman admin dan superadmin

1
Buka Halaman Login
Akses halaman login melalui browser dengan membuka URL sistem. Masukkan email dan password sesuai akun yang telah diberikan.
  • Buka browser dan akses URL sistem
  • Masukkan Email dan Password pada form login
  • Klik tombol Masuk
  • Sistem akan mengarahkan ke dashboard sesuai role (Admin / SuperAdmin)
💡

Jika lupa password, hubungi SuperAdmin untuk mereset akun Anda.

Admin Restoran — Dashboard
📊

Dashboard Admin ADMIN

Halaman utama yang menampilkan ringkasan data dan analitik restoran

1
Ringkasan Statistik
Dashboard menampilkan total pesanan, total pendapatan, jumlah menu, dan notifikasi stok rendah.
2
Grafik Penjualan
Dashboard menampilkan grafik tren penjualan harian untuk memudahkan monitoring performa restoran.
3
Perencanaan Stok (Regresi Linear)
Tabel perencanaan stok menggunakan algoritma regresi linear untuk memprediksi kebutuhan stok berdasarkan histori penjualan.
  • Pilih periode analisis (7, 14, 30, atau 60 hari)
  • Sistem menghitung persamaan regresi Y = a + bX per menu
  • Kolom Status menunjukkan apakah stok perlu ditambah atau masih aman
ℹ️

Periode 7 dan 14 hari mungkin tidak menampilkan hasil jika data penjualan belum cukup untuk dihitung secara statistik.

Admin Restoran — Manajemen Kategori
🏷️

Kelola Kategori Menu ADMIN

Menambah, mengedit, dan menghapus kategori untuk pengelompokan menu

1
Melihat Daftar Kategori
Buka menu Kategori di sidebar untuk melihat semua kategori yang tersedia di restoran Anda.
2
Tambah Kategori Baru
Klik tombol + Tambah Kategori, isi nama kategori, lalu simpan.
3
Edit / Hapus Kategori
Klik ikon Edit untuk mengubah nama, atau ikon Hapus untuk menghapus kategori.
⚠️

Menghapus kategori yang masih memiliki menu aktif tidak diperbolehkan. Pindahkan atau hapus menu terlebih dahulu.

Admin Restoran — Manajemen Menu
🍽️

Kelola Menu ADMIN

Menambah, mengedit, mengatur stok dan harga produk menu restoran

1
Daftar Menu
Halaman menu menampilkan semua produk beserta harga, stok, kategori, dan status (aktif/nonaktif). Gunakan filter kategori untuk menyaring tampilan.
2
Tambah Menu Baru
Klik + Tambah Menu. Isi nama, deskripsi, harga, stok, pilih kategori, dan upload foto menu (maks. 2MB).
  • Isi Nama Menu dengan jelas dan spesifik
  • Tentukan Harga dalam Rupiah
  • Isi Stok yang tersedia
  • Pilih Kategori yang sesuai
  • Upload Foto Menu (format: JPG/PNG, maks 2MB)
  • Klik Simpan
💡

Setelah menyimpan, sistem akan otomatis mengarahkan ke halaman daftar menu dengan filter kategori yang sesuai.

3
Edit Menu
Klik ikon Edit pada menu yang ingin diubah. Ubah informasi yang diperlukan lalu klik Simpan Perubahan.
Admin Restoran — Manajemen Pesanan
🧾

Kelola Pesanan ADMIN

Memonitor dan memperbarui status pesanan dari pelanggan secara real-time

1
Daftar Pesanan
Halaman pesanan menampilkan semua order yang masuk, lengkap dengan nomor meja, status pembayaran, dan total harga.
2
Detail Pesanan
Klik tombol Detail untuk melihat rincian item yang dipesan oleh pelanggan beserta jumlah dan harga satuan.
3
Update Status Pesanan
Ubah status pesanan sesuai progres (Diproses → Siap → Selesai). Admin akan menerima notifikasi otomatis saat pesanan baru masuk.
  • Buka detail pesanan yang ingin diubah statusnya
  • Pilih status baru dari dropdown yang tersedia
  • Klik Update Status
  • Pelanggan akan menerima notifikasi perubahan status
4
Export Data Pesanan
Data pesanan dapat diekspor ke format Excel/CSV untuk keperluan laporan keuangan.
  • Atur filter tanggal sesuai rentang yang diinginkan
  • Klik tombol Export
  • File akan otomatis terunduh ke komputer Anda
Admin Restoran — Meja & QR Code
🪑

Kelola Meja & QR Code ADMIN

Mengatur nomor meja dan menghasilkan kode QR untuk pemesanan mandiri pelanggan

1
Daftar Meja
Halaman ini menampilkan semua meja yang terdaftar beserta status dan QR code masing-masing.
2
Tambah Meja
Klik + Tambah Meja, masukkan nomor meja, dan sistem akan otomatis menghasilkan QR Code unik.
3
Download QR Code
Unduh QR Code setiap meja untuk dicetak dan dipasang di meja restoran agar pelanggan dapat scan untuk memesan.
Admin Restoran — Profil
👤

Edit Profil Restoran ADMIN

Memperbarui informasi dan identitas restoran yang tampil ke pelanggan

1
Ubah Informasi Restoran
Perbarui nama restoran, deskripsi, nomor telepon, alamat, dan foto/logo restoran melalui halaman Edit Profil.
  • Buka menu Profil / Restoran di sidebar
  • Ubah informasi yang diperlukan pada form
  • Upload foto restoran baru jika ingin mengganti
  • Klik Simpan Perubahan
SuperAdmin — Dashboard
⚡

Dashboard SuperAdmin SUPERADMIN

Ringkasan keseluruhan sistem: total restoran, admin, dan statistik platform

1
Halaman Dashboard SuperAdmin
Setelah login sebagai SuperAdmin, Anda akan melihat ringkasan jumlah restoran aktif, total admin, dan statistik platform secara keseluruhan.
SuperAdmin — Manajemen Restoran
🏪

Kelola Restoran SUPERADMIN

Menambah, mengedit, menghapus, dan memantau semua restoran di platform

1
Daftar Semua Restoran
Menu Restoran menampilkan seluruh restoran yang terdaftar dalam platform beserta informasi singkat tiap restoran.
2
Tambah Restoran Baru
Klik + Tambah Restoran untuk mendaftarkan restoran baru ke dalam platform.
  • Isi Nama Restoran, alamat, dan nomor telepon
  • Upload Logo/Foto Restoran
  • Isi Email dan Password untuk akun admin restoran tersebut
  • Klik Simpan — sistem otomatis membuat akun admin baru
3
Edit Data Restoran
Klik ikon Edit pada baris restoran untuk memperbarui informasi nama, alamat, telepon, atau logo.
4
Lihat Menu Restoran
Klik tombol Lihat Menu pada salah satu restoran untuk melihat seluruh daftar menu yang dimiliki restoran tersebut.
5
Hapus Restoran
Klik ikon Hapus untuk menghapus restoran dari platform. Aksi ini bersifat permanen.
⚠️

Menghapus restoran akan menghapus seluruh data terkait (menu, pesanan, dll). Pastikan data sudah diarsipkan sebelum menghapus.

SuperAdmin — Manajemen Admin
👥

Kelola Akun Admin SUPERADMIN

Menambah, menonaktifkan, dan memulihkan akun admin restoran

1
Daftar Admin
Halaman ini menampilkan semua akun admin yang terdaftar di platform beserta status aktif/nonaktif.
2
Tambah Admin Baru
Klik + Tambah Admin, isi nama, email, password, dan pilih restoran yang dikelola.
3
Edit Akun Admin
Klik Edit untuk memperbarui nama, email, password, atau restoran yang dikelola admin.
4
Nonaktifkan & Pulihkan Admin
Admin dapat dinonaktifkan (soft delete) tanpa menghapus data. Akun yang dinonaktifkan dapat dipulihkan kembali kapan saja.
  • Klik Nonaktifkan untuk menonaktifkan akun admin (data tetap tersimpan)
  • Akun yang dinonaktifkan akan muncul di daftar dengan status Tidak Aktif
  • Klik Pulihkan untuk mengaktifkan kembali akun admin
💡

Gunakan fitur Nonaktifkan daripada Hapus Permanen agar data histori pesanan tetap tersimpan.
